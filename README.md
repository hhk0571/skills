## 📚 简介

这是一个个人维护的 AI Skills 仓库，包含一系列专业领域的 AI 技能模块。每个 Skill 都经过精心设计和测试，可以直接用于 Claude AI 或其他支持 Skills 的 AI 助手。

## 🎯 Skills 列表

### 1. 婴儿取名 (Baby Naming) 🍼

基于中华传统五行文化的智能婴儿取名助手，支持八字计算、五行分析、名字推荐和评估。

- **文件**: `baby-naming/`
- **功能**:
  - 精准八字计算和五行分析
  - 智能推荐寓意美好的名字
  - 名字评估打分（100分制）
  - 多维度评价（五行匹配、文化内涵、音韵美感等）
- **依赖**: Python 3.6+, lunar-python (推荐)
- **详细说明**: [查看 README](baby-naming/README.md)

## 🚀 快速开始

### 安装 Skills

```bash
# 方式1：使用 npx skills 安装
npx skills add hhk0571/skills

# 方式2：克隆仓库
git clone https://github.com/hhk0571/skills.git
cd skills
```

### 使用 Skill

每个 Skill 都包含详细的使用说明，请查看对应的 README 文档。

## 📖 Skill 结构

每个 Skill 通常包含以下文件：

```
skill-name/
├── README.md           # 详细使用说明
├── SKILL.md           # Skill 核心定义
├── scripts/           # 辅助脚本（如有）
└── references/        # 知识库和参考资料（如有）
```
