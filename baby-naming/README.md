## 📖 简介

这是一个基于**中华传统文化**和**阴阳五行理论**的 AI 婴儿取名专家 Skill。它可以根据婴儿的生辰八字（出生年月日时）、性别和姓氏，结合五行理论和中华传统文化，智能推荐寓意美好、五行匹配的名字，或评估已有名字的匹配度。

## ✨ 功能特点

- **🔮 精准八字计算** - 支持专业 lunar_python 库，提供最准确的八字和五行分析
- **⚖️ 五行平衡分析** - 自动识别缺失、偏弱、偏强的五行，智能推荐匹配名字
- **📝 智能推荐名字** - 根据八字特征推荐 5-10 个寓意美好、五行协调的名字
- **📊 名字评估打分** - 对已有名字进行全面评估，提供详细的评分报告（100分制）
- **🎨 多维度评价** - 从五行匹配、文化内涵、音韵美感、字形结构、性别特征等多维度评价
- **🔄 自动降级机制** - 如果未安装专业库，自动使用简化算法，确保基础功能可用
- **📚 丰富知识库** - 内置五行理论、常用字库、命名原则、评估标准等完整知识体系

## 🚀 快速开始

### 安装 Skill

```bash
# 方式1：直接复制到 skills 目录
cp baby-naming ~/.claude/skills/

# 方式2：使用 openskills 加载
npx openskills read baby-naming
```

### 安装依赖（强烈推荐）

为了获得最准确的八字计算结果，强烈建议安装 `lunar-python` 库：

```bash
pip install lunar-python
```

> **说明**：
> - ✅ 安装后，将使用专业的 `lunar_python` 库进行八字计算，精确度更高
> - ⚠️ 如果未安装，脚本会自动降级使用简化算法，但可能存在误差
> - 📦 `lunar_python` 是一个成熟的农历和八字计算库，由 [6tail](https://github.com/6tail/lunar-python) 开发维护

## 🎯 核心功能详解

### 1. 八字计算脚本

**特性：**
- ✅ 优先使用 `lunar_python` 库进行精确计算
- ✅ 自动降级机制：库不可用时使用简化算法
- ✅ 分析五行属性（木、火、土、金、水）
- ✅ 识别缺失、偏弱、偏强的五行
- ✅ 确定日主属性
- ✅ JSON 格式输出，易于处理

**使用方法：**
```bash
# 计算八字（年 月 日 时）
python scripts/bazi_calculator.py 2026 1 27 10 --json
```

### 2. 评分标准

总分 **100 分**，包含五个维度：

| 维度 | 分值 | 说明 |
|------|------|------|
| **五行匹配度** | 30分 | 与八字的五行匹配程度，是否补足缺失或偏弱的五行 |
| **文化内涵** | 25分 | 寓意美好程度、是否有诗词典故、文化底蕴 |
| **音韵美感** | 20分 | 平仄搭配、读音流畅度、声调和谐 |
| **字形结构** | 15分 | 书写方便、结构美观、笔画适中 |
| **性别特征** | 10分 | 是否符合性别特征，避免男女不分 |

### 3. 五行理论

**相生关系：**
```
木 → 火 → 土 → 金 → 水 → 木
```

**相克关系：**
```
木克土、土克水、水克火、火克金、金克木
```

**命名原则：**
- 补足缺失的五行
- 加强偏弱的五行
- 形成相生关系，避免相克
- 整体平衡协调

## ⚙️ 技术细节

### 依赖

- Python 3.6+
- **lunar-python** (推荐): `pip install lunar-python`
  - 提供精确的八字计算
  - 如果未安装，脚本会自动使用简化算法

### 八字计算说明

- 基于公历（阳历）时间
- 需要精确到小时
- **推荐使用 lunar_python 库**：提供最准确的八字计算
- 自动降级机制：如果库不可用，使用简化算法
- lunar_python 使用传统的干支纪年法和农历转换算法

## 💡 使用建议

1. **准确性**：需要准确的出生时辰（精确到小时）
2. **平衡性**：不过度强调单一五行，注重整体协调
3. **文化性**：名字体现中华文化的美好寓意
4. **实用性**：避免生僻字和过于复杂的字
5. **个性化**：根据具体情况灵活调整

## ❓ 常见问题

<details>
<summary><b>Q: 如果不知道准确的出生时辰怎么办？</b></summary>

A: 尽量获取准确时间。如果只知道大概时间段（如上午、下午），可以取中间值（如上午10点、下午3点）。
</details>

<details>
<summary><b>Q: 五行缺失一定要补吗？</b></summary>

A: 不一定。主要看整体平衡和日主强弱。有时缺失的五行不需要补，而是需要加强偏弱的五行。
</details>

<details>
<summary><b>Q: 评分高的名字就一定好吗？</b></summary>

A: 评分是综合参考，但最终选择还要考虑家庭喜好、长辈意见等因素。
</details>

<details>
<summary><b>Q: 可以只提供八字结果而不提供出生时间吗？</b></summary>

A: 可以。如果已经知道八字和五行情况，可以直接提供，不需要重新计算。
</details>

<details>
<summary><b>Q: 为什么推荐安装 lunar_python 库？</b></summary>

A: lunar_python 是一个成熟的专业库，能提供更准确的八字计算，考虑节气、真太阳时等因素。简化算法虽然可用，但可能存在误差。
</details>
